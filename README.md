# Gowsik B - Portfolio

Welcome to my personal portfolio website! This is a modern, responsive, and dynamic web application showcasing my skills, projects, achievements, and professional journey.

## 🚀 Live Demo & Repository
- **GitHub Repository**: [https://github.com/GOWSIK-977/Portfolio](https://github.com/GOWSIK-977/Portfolio)

## ✨ Features
- **Modern Dark Aesthetics**: Custom CSS styling with HSL tailored color schemes, glassmorphism, dynamic shadows, and glowing accent highlights.
- **Dynamic Particle Background**: Smooth floating particle effects rendering in the background.
- **Parallax Mouse Interaction**: Profile card and floating badges that dynamically follow mouse movement.
- **Typing Animation**: Looping typing effect highlight for roles ("MERN Stack Developer | IoT Enthusiast | AI Explorer").
- **Interactive Project Cards**: Customized accent-colored grids linking directly to GitHub projects.
- **Milestones & Achievements**: Detailed chronological view of hackathons (SIH 2025, Balance-Bytes, Byte-Rush) and academic performance.
- **Responsive Navigation & Smooth Scroll**: Full navigation control with sticky/backdrop navbar and optimized scroll indicators.
- **Contact Form**: Responsive message box with micro-animations and inline input icons.

## 🛠️ Tech Stack
- **Frontend**: React (v18), JavaScript (ES6+), HTML5, Vanilla CSS
- **Build Tool**: Vite
- **Icons**: Font Awesome (CDN & Free-package integration)
- **Fonts**: Inter (Google Fonts)

## 📦 Getting Started

To run this project locally, follow these steps:

### Prerequisites
Make sure you have [Node.js](https://nodejs.org) installed on your system.

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/GOWSIK-977/Portfolio.git
   cd Portfolio
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the local development server:
   ```bash
   npm run dev
   ```

4. Build for production:
   ```bash
   npm run build
   ```
